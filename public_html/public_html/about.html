About This Project

Project Overview

The "Refreshment 3D" application was developed as part of the Mobile 3D App Assignment. This interactive web application showcases three beverage container models: a cola can, a plastic bottle, and a glass bottle. Users can interact with these 3D models, view them in wireframe mode, control lighting, and learn about each product.

Technologies Used

Frontend Technologies

  • HTML5: For structuring the web pages
  • CSS3: For styling and responsive design
  • Bootstrap 5: For responsive layout and components
  • JavaScript: For interaction and functionality

3D Technologies

  • Three.js: JavaScript 3D library for rendering the models
  • Blender: For creating and editing the 3D models
  • glTF/GLB: For optimized 3D model delivery
  • OrbitControls: For camera movement and interaction
  • Animation Mixer: For handling model animations

3D Models

All 3D models were created using Blender. Special attention was paid to geometry, materials, and textures to create realistic beverage containers:

Cola Can

The cola can model was created using cylinder primitives with edge loops for the top and bottom details. Custom textures were applied to create the label and metallic surface.

Polygon count: Approximately 2,400 triangles

Animation states: Default, open (ring pull), and crush simulation

Plastic Bottle

The plastic bottle was modeled using a combination of revolve operations and subdivision surface modifiers. The cap was modeled separately and includes threading details.

Polygon count: Approximately 8,600 triangles

Animation states: Default, open cap, empty, and crush simulation

Glass Bottle

The glass bottle features a complex curved shape created using spline-based modeling. Glass materials use PBR (Physically Based Rendering) for realistic refraction and reflection.

Polygon count: Approximately 12,800 triangles

Animation states: Default, open cap, empty, and crush simulation

Interactive Features

Model Interaction

  • Orbit controls for 3D navigation
  • Toggle wireframe mode
  • Toggle automatic rotation
  • Adjust lighting conditions
  • Switch between different models
  • Play sound effects for model interactions
  • Camera positioning controls (top, bottom, left, right, front, back)
  • Screenshot functionality

User Interface

  • Responsive design for mobile and desktop
  • Model selection panel
  • Interactive control buttons
  • Product information display
  • Multi-model gallery view
  • Adjustable model scale and rotation speed
  • Advanced control panel with interactive sliders

Audio Implementation

The application features realistic sound effects for various interactions:

  • Opening sounds: Unique audio cues for opening cans, plastic bottles, and glass bottles
  • Crushing sounds: Distinct sounds for each material when crushed
  • Volume control: User-adjustable audio levels
  • Mute functionality: Option to completely disable sound

Development Process

The development of this project followed these key steps:

  1. Planning: Defining requirements and sketching the UI/UX design
  2. 3D Modeling: Creating detailed models in Blender with appropriate textures
  3. Animation: Rigging and animating the models for different states
  4. Audio: Recording and editing sound effects for various interactions
  5. Web Development: Building responsive HTML/CSS layouts with Bootstrap
  6. 3D Integration: Implementing Three.js for model rendering and interaction
  7. Testing: Cross-browser testing and mobile responsiveness
  8. Optimization: Refining model complexity and texture sizes for web delivery
  9. User Testing: Gathering feedback and making improvements

Challenges and Solutions

Challenges

  • Creating realistic materials for glass and liquid
  • Optimizing models for web performance
  • Implementing responsive design for 3D content
  • Managing lighting to showcase model details
  • Synchronizing animations with sound effects
  • Creating realistic crush animations
  • Supporting wireframe mode with animations
  • Managing memory for multiple 3D models in gallery view

Solutions

  • Used PBR materials with optimized texture sizes
  • Reduced polygon count while preserving visual quality
  • Implemented responsive container sizing for all devices
  • Added interactive lighting controls
  • Created custom event listeners for animation/sound synchronization
  • Used shape keys in Blender for realistic deformations
  • Developed custom wireframe handling with THREE.WireframeGeometry
  • Implemented efficient scene management and disposal

Performance Optimization

Several techniques were employed to optimize the application's performance:

  • Model simplification: Reduced polygon count while maintaining visual fidelity
  • Texture compression: Optimized image sizes and formats
  • Lazy loading: Models are only loaded when selected
  • Efficient animation: Using THREE.AnimationMixer for optimized playback
  • Memory management: Proper cleanup of unused 3D objects
  • Responsive rendering: Adjusting render quality based on device capabilities

User Testing Insights

The application was tested with several users to gather feedback and make improvements:

  • Users appreciated the realistic model details and animations
  • Mobile controls were initially challenging for some users, leading to interface improvements
  • Sound effects were consistently rated as adding value to the experience
  • The gallery view was preferred for comparing different container types
  • Users suggested additional features like AR viewing capabilities

Learning Outcomes

Key skills and knowledge gained from this project:

  • Advanced Three.js implementation for interactive 3D web applications
  • Optimization techniques for web-based 3D content
  • Integration of sound with 3D animations
  • Responsive design principles for 3D interfaces
  • User experience considerations for 3D interactions
  • Cross-browser compatibility for WebGL content

Conclusion

The 3D Beverage Showcase represents a successful integration of 3D modeling, web technologies, and interactive design. By creating a responsive, feature-rich application that allows users to explore and interact with beverage containers in three dimensions, the project demonstrates how 3D content can enhance product visualization and user engagement.

The challenges overcome during development—from creating realistic materials to optimizing performance across devices—provided valuable learning experiences in both technical and design aspects of 3D web applications. The integration of sound effects with animations adds another sensory dimension to the user experience, making interactions more engaging and intuitive.

This project serves as both an educational tool for understanding beverage container design and a demonstration of what's possible when combining modern web technologies with 3D content. The responsive design ensures accessibility across a wide range of devices, making this application a versatile showcase for 3D web development.
