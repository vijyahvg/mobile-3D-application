Original Work

The following components of this project represent my original work:

  • 3D modeling: All beverage container models (can, plastic bottle, glass bottle) were created from scratch in Blender
  • Animation: All model animations including opening, emptying, and crushing effects
  • HTML/CSS structure: Custom layout and styling of the website using Bootstrap framework
  • JavaScript implementation:
    • Custom implementation of Three.js for model loading and rendering
    • User interface controls for model interaction
    • Animation system for model state changes
    • Sound effect integration with animations
    • Camera control system with multiple viewpoints
    • Custom wireframe implementation
  • Sound effects: Recording and editing of all sound effects for model interactions
  • Website design: Responsive layout for desktop and mobile devices
  • Content: All product descriptions, informational content, and comparison data

Development Methodology

This project was developed using an iterative approach with the following stages:

  1. Research and Planning: Investigating Coca-Cola product designs and three.js capabilities
  2. Design Phase: Creating wireframes and mockups for website layout
  3. 3D Modeling: Developing detailed models in Blender with appropriate materials
  4. Web Framework: Building responsive Bootstrap layout
  5. Integration: Connecting Three.js with the web interface
  6. Feature Implementation: Adding interactive elements, animations, and controls
  7. Testing and Refinement: Optimizing for performance and fixing issues
  8. Documentation: Completing project documentation and statements

The development process utilized Git for version control, with regular commits to track progress and allow for rollbacks when necessary.

Learning Reflection

This project has significantly enhanced my technical skills and understanding in several areas:

  • 3D Web Development: I gained practical experience implementing Three.js for interactive 3D content on the web, learning how to balance visual quality with performance considerations.
  • Responsive Design: Creating a 3D application that works well across devices presented unique challenges, especially for touch interfaces, which improved my understanding of responsive design principles.
  • 3D Modeling: Developing detailed models with different materials (metal, plastic, glass) enhanced my Blender skills, particularly in creating optimized models for web delivery.
  • Animation: Implementing animation systems for model interactions taught me about keyframe animation, shape keys, and programmatic animation control.
  • Sound Integration: Adding synchronized sound effects to 3D interactions improved my understanding of web audio APIs and event-driven programming.
  • User Experience Design: Through testing and iteration, I developed a better understanding of how users interact with 3D content and how to design intuitive controls.

This project also highlighted the importance of continuous learning and problem-solving in web development. Each challenge encountered—from optimizing model loading to ensuring proper wireframe rendering during animations—required research, experimentation, and creative solutions. These experiences have significantly improved my ability to tackle complex technical challenges in future projects.
